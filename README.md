## 📦 Installation

1. Clone the repository
```bash
git clone https://github.com/Devarajb049/Dynamic-Portfolio
```

2. Navigate to the project directory
```bash
cd Dynamic-Portfolio
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```
